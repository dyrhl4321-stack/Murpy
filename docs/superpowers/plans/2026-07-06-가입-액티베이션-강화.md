# 가입 액티베이션 강화 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 온보딩에 프로필 사진 필수 단계를 추가하고, 가입 완료 시 머피(+10) 축하·설명 가이드를 띄워 유저를 핵심 기능(매칭·대숲)으로 안착시킨다. 머피월드로 강제 이동 없음.

**Architecture:** 기존 `#onboarding-modal`(2스텝)에 Step3(프로필 사진)를 추가하고, 기존 프로필 크롭 플로우(`profile-pic-input`→`openProfileCrop`→`confirmProfileCrop`→`_pendingProfilePhoto`+`save-profile-photo`)와 기존 머피 연출(`celebrateMurpy`, `#murpy-first-guide`)을 **재활용**한다. 새 컴포넌트 최소.

**Tech Stack:** 단일 `index.html`, 쌩 JS, Firebase Firestore, imgBB. 테스트 프레임워크 없음 → 로컬 브라우저 육안 + `node --check`.

## Global Constraints

- 대상 파일: `C:\Users\won\Murpy\index.html` (단일 HTML).
- **핵심 기능 = 매칭·피드·대숲.** 머피월드는 서브월드 → 가입 직후 **머피월드로 강제 이동 금지.** "머피월드 가실래요?" 류 X.

- [ ] **Step 1: obComplete 교체 (사진 게이트 + 축하/가이드)**

`index.html:7361-7382`의 `window.obComplete = async () => { ... };` 전체를 교체. 종목·지역 검증은 obNextStep2로 이동했으므로 제거하고, 사진 게이트 + 저장 후 축하/가이드 추가:
```javascript
window.obComplete = async () => {
  // 종목·지역은 obNextStep2에서 검증됨. 여기선 프로필 사진 필수 게이트.
  if (!window._pendingProfilePhoto && !(window.currentUser && window.currentUser.firestorePhotoURL)) {
    showToast('프로필 사진을 등록해주세요'); return;
  }
  const district = document.getElementById('ob-district').value;
  const user = auth.currentUser || window.currentUser;
  if (!user) return;
  const nickname = document.getElementById('ob-nickname').value.trim();
  const age = parseInt(document.getElementById('ob-age').value);
  const bio = document.getElementById('ob-bio').value.trim();
  try {
    await setDoc(doc(db, 'users', user.uid), {
      nickname, age, gender: window._obData.gender,
      purpose: window._obData.purpose, workoutTime: window._obData.times,
      sports: window._obData.sports,
      district, bio, onboardingComplete: true, updatedAt: serverTimestamp()
    }, { merge: true });
    if (window.currentUser) window.currentUser.nickname = nickname;
    document.getElementById('onboarding-modal').style.display = 'none';
    await window.loadUserProfile(user.uid);
    // 가입 선물: 머피 축하 연출 + 설명 가이드(핵심 기능으로 유도). 1회.
    if (window.celebrateMurpy) window.celebrateMurpy(window.getCredits ? window.getCredits() : 10);
    try { localStorage.setItem('murpy_credit_intro', '1'); } catch (e) {}
    setTimeout(function () { if (window.murpyFirstGuide) window.murpyFirstGuide(); }, 2700);
  } catch(e) { console.error('온보딩 저장 실패:', e); showToast('저장 실패: ' + e.message); }
};
```
