# 즉석 스쿼드 P0 (실시간 워킹 룸 포함) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** '크루' 탭을 '스쿼드'로 전환하고, 벙주(호스트)용 30초 개설 → 참가 → 실시간 워킹 룸(멤버들이 종목 필드에서 같이 걸어다님) → 채팅 → QR/코드 출석 → 자동 종료 + 호스트 머피 적립까지 P0 전체를 웹으로 구현한다.

**Architecture:** 단일 `index.html`. 영속 데이터=Firestore(`squads` 컬렉션), 휘발 위치=**Firebase Realtime Database**(`squadRooms`, 신규). 이동·캐릭터 렌더는 기존 머피월드(`mwMiniCharHtml`, CSS transition 보간) 재사용. UI 위계: 스쿼드 탭·카드=앱 레벨 라운드 스타일, 워킹 룸 내부=머피월드 픽셀 스타일.

**Tech Stack:** HTML/CSS/JS 단일 파일, Firebase Auth/Firestore/RTDB(10.12.0 ES module), GitHub Pages, sw.js 캐시.

**스펙:** `docs/superpowers/specs/2026-07-19-squad-p0-design.md`

## Global Constraints

- 모든 변경은 `index.html`+`sw.js`+`firestore.rules`. sw `murpy-vNNN` 3곳 버전업(현재 **v167**), sw.js는 Edit 도구로만.
- 편집 후 매 태스크 `node tools/dogam-syntax-check.mjs` 통과 필수.
- 기존 크루 코드·데이터 **삭제 금지**(휴면). `page-crew` DOM id는 재사용, 렌더 진입만 교체.
- Firebase 모듈 함수는 module 스코프 → onclick에서 쓸 함수는 전부 `window.*` 브릿지.
- 색: 블루 #3D7EFF=액션, 골드 #F5C24B=보상. 이모지 금지(라인 SVG). 픽셀 아이콘·픽셀 프레임은 워킹 룸 안에서만.
- 종목 7종 키: `gym|running|golf|tennis|climbing|hyrox|hiking` → 필드 `char/fields/field_<key>_wide.png?v=1`(gym만 `?v=2`).
- 커밋 메시지 한글 → Bash(Git Bash) `-m` heredoc 또는 `-F`. Co-Authored-By 유지.
- **RTDB 미활성 폴백:** RTDB 연결 실패 시 워킹 룸은 정적 배치로 폴백(`window._SQUAD_RT_OFF=true`). 배포가 콘솔 작업에 블로킹되지 않게.

## UX "와우" 포인트 (태스크에 분산 반영)

1. **카드 썸네일 = 종목 와이드 필드 이미지** (신규 에셋 0개로 게임 카드 느낌).
2. **워킹 룸**: 탭하면 캐릭터가 걸어감 + **다른 멤버가 실시간으로 움직임** + **채팅이 캐릭터 머리 위 말풍선**으로 뜸 + 입장 시 하단에서 걸어 들어오는 연출 + "OO님이 들어왔어요" 시스템 메시지.
3. **출석 도장 팝**: 체크인 순간 그 캐릭터 머리 위에 골드 도장 스탬프가 쾅 찍히는 이펙트(룸 안 전원에게 실시간으로 보임).
4. 채팅 = 센터 대숲에서 확정한 게임식 하단 입력 + column-reverse 패턴 재사용.

```js
window.joinSquad = async function (sid) {
  const user = auth.currentUser; if (!user) return requireLogin();
  try {
    await runTransaction(db, async tx => {
      const sRef = doc(db,'squads',sid); const s = await tx.get(sRef);
      if (!s.exists()) throw new Error('없는 스쿼드');
      const d = s.data();
      if ((d.memberUids||[]).includes(user.uid)) throw new Error('이미 참가중');
      if (d.joinMode==='instant' && (d.memberUids||[]).length >= d.capacity) throw new Error('정원 마감');
      if (d.joinMode==='instant') tx.update(sRef, { memberUids:[...d.memberUids,user.uid] });
      tx.set(doc(db,'squads',sid,'members',user.uid), {
        status: d.joinMode==='instant'?'accepted':'invited',
        nickname:window._mwMyNick||'머피',
        character:window.getMyCharacter?window.getMyCharacter():null, joinedAt:serverTimestamp() });
    });
    showToast('참가 완료!');
  } catch(e) { showToast(e.message||'참가 실패'); }
};
```
승인 모드: 호스트에게 `invited` 목록 + `[승인]` → `approveSquadMember(sid,uid)`가 memberUids 추가+status accepted(같은 트랜잭션 패턴). `leaveSquad` = memberUids 제거 + status left.

- [ ] **Step 1: 입장/쓰기**

```js
window._sqRtJoin = function (sid) {
  if (window._SQUAD_RT_OFF || !window._rt) return;   // 폴백: 정적 배치(멤버는 Task 3 리스트로만)
  const u = auth.currentUser; if (!u) return;
  const R = window._rt, me = R.ref(R.db, `squadRooms/${sid}/players/${u.uid}`);
  R.onDisconnect(me).remove();
  const send = () => R.set(me, { x:window._sqMyPos.x, y:window._sqMyPos.y,
    dir:window._sqMyPos.dir, nick:window._mwMyNick||'머피',
    character:window.getMyCharacter?window.getMyCharacter():null, t:Date.now() });
  send();
  window._sqOnMove = (() => { let last=0; return () => {           // 쓰로틀 220ms
    const n=Date.now(); if (n-last>220) { last=n; send(); }
    clearTimeout(window._sqMoveT); window._sqMoveT=setTimeout(send,260); }; })();
  window._sqRtUnsub = R.onValue(R.ref(R.db, `squadRooms/${sid}/players`), snap => {
    const all = snap.val()||{}; window._sqRenderRemotes(sid, all, u.uid);
  });
};
window._sqRtLeave = function (sid) {
  const u=auth.currentUser; if (window._rt && u) try { window._rt.remove(window._rt.ref(window._rt.db,`squadRooms/${sid}/players/${u.uid}`)); } catch(e){}
  window._sqRtUnsub && window._sqRtUnsub(); window._sqRtUnsub=null; window._sqOnMove=null;
};
```
