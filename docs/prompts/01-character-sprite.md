# 01 · 캐릭터 스프라이트 시트 (12칸 걷기)

3열×4행 = 12칸. **행:** 정면(아래)/후면(위)/좌측/우측, **열:** 정지/걷기A/걷기B.
기본 캐릭터·헬토리가 이 규격. 새 고정 캐릭터(히든 캐릭터 등)도 동일.

> 먼저 [공통 규칙](README.md#모든-프롬프트-공통-규칙-나노바나나-하드룰)을 읽을 것.

## 워크플로우
1. 레퍼런스 이미지 **1장** 첨부(구조·톤 기준).
   - ⚠️ **헬토리를 레퍼런스로 쓰면 이상하게 나온다 → '빡빡이(기본 체형)' 권장.**
   - 단 빡빡이는 민머리·헐벗음이라, 아래 공통 블록의 **[아주 중요] 오버라이드**가 그걸 막는다.
2. 아래 **[공통 블록] + [캐릭터 블록]** 을 이어 붙여 생성.
3. 뽑히면 [검수 체크리스트](#검수-체크리스트) 확인 → ✦ 있으면 워터마크 제거 툴 → Photoroom 누끼.

---

## [공통 블록] — 캐릭터마다 앞에 붙인다
```
첨부한 기본 캐릭터 도트 스프라이트 시트를 '체형·비율·도트 스타일·12칸 시트 구조'의
참고용으로만 써서, 아래 [캐릭터]를 12칸 도트 스프라이트 시트로 새로 그려줘.

[아주 중요] 레퍼런스는 민머리·기본차림의 '기본 체형'일 뿐이다. 그 민머리·헐벗음을 절대 따라가지 마.
 → 캐릭터는 반드시 (1) 머리카락(또는 머리 덮개)을 갖추고 (2) 상·하의를 완전히 갖춰 입은 상태로.
   대머리 금지, 상의탈의·헐벗음 금지. 아래 설명의 머리·의상을 확실하게 입혀서 그릴 것.

[구조] 3열×4행=12칸, 칸 크기·간격 동일 / 행: 1행 정면 2행 후면 3행 좌측 4행 우측 /
 열: 1열 정지 2열 걷기A 3열 걷기B / 좌우측면 미러(좌우반전) 금지(각각 실제 방향) /
 모든 칸 크기·발바닥선·중심 정렬 동일 / 픽셀 아트, 선명한 계단식 도트(안티에일리어싱·블러 금지).
[소품 규칙] 앞뒤옆이 확 다른 입체 소품(가방·손에 든 물건) 금지 — 명시된 납작한 요소만.
[배경/출력] 단색 형광초록 #00FF00(누끼용, 투명 금지). 워터마크·반짝이(✦)·불필요한 글자·그림자 없음.
[캐릭터]
```

## [캐릭터 블록] 예시 (실제 사용본)

### 헬토리 (근방단 한정 · 백팩 뺀 미니멀)
```
'헬토리' — 흰토끼 헬창 캐릭터.
- 백팩·물통 없음(손에 든 것, 등에 멘 것 모두 없음).
- 셔츠 가슴에 "BORN TO WIN" + GBD 로고를 납작하게 프린트. 후면은 등판에 같은 로고. 그 외 소품 없음.
- 흰 토끼 귀 유지.
```
> 백팩을 뺀 이유: 가방은 입체라 방향마다 크기·모양이 어긋났다(정면=크고 후면=작음, 물통 좌우 뒤집힘).

### 히든 · 화요일교 교주
```
'화요일교 교주' — 능글맞고 신비로운 사이비 교주 느낌의 치비 인간 캐릭터.
- 머리: 두건 또는 소박한 관으로 머리를 덮음(민머리 노출 금지). 머리 뒤에 납작한 원형 후광(은은한 금빛).
- 의상: 발목까지 오는 헐렁한 로브를 완전히 착용(상·하체 전부 덮임). 팔·다리는 걷기 동작 되게.
- 표정: 다 안다는 듯 지그시 웃는 눈. 손에 든 지팡이·물건 없음(후광·두건·로브만).
```

### 히든 · 헬스장 소믈리에
```
'헬스장 소믈리에' — 헬스장을 시음하는 미식가 컨셉의 댄디한 치비 인간 캐릭터.
- 머리: 단정하게 빗어넘긴 짧은 머리(민머리 금지).
- 의상: 셔츠 + 나비넥타이 + 가슴~허리 앞치마 + 긴바지, 상·하의 완전 착용(헐벗음 금지).
  앞치마에 와인잔 엠블럼을 납작하게 프린트.
- 표정: 살짝 콧수염, 지그시 감식하는 여유로운 표정. 손에 든 와인잔·물건 없음.
```

### 히든 · 작심삼일 좀비
```
'작심삼일 좀비' — 3일마다 죽었다 살아나는 헬스 좀비. 킹받게 귀여운 치비 인간 캐릭터.
- 머리: 헝클어진 머리(민머리 금지), 반쯤 풀린 눈 + 다크서클.
- 피부: 창백하고 살짝 초록빛.
- 의상: 해진(찢어진) 운동 셔츠 + 바지 완전 착용, 몸 곳곳에 납작한 붕대. 헐벗음·상의탈의 금지.
- 표정/동작: 영혼 없이 터덜터덜(걷기 프레임에서 흐느적). 안 무섭고 웃긴 느낌. 손에 든 것 없음.
```

> **몸 베이스:** 위 히든 3종은 '치비 인간' 기준. 헬토리처럼 토끼로 통일하려면 각 설명에 `토끼 귀 유지` 한 줄 추가.

## 검수 체크리스트
- [ ] 12칸 다 있고, 칸마다 캐릭터 크기·발바닥선·중심 일정
- [ ] 좌·우측면이 미러가 아니라 각각 실제 방향
- [ ] 대머리·헐벗음 없음 (머리·상하의 완비)
- [ ] 입체 소품(가방·든 물건) 없음
- [ ] 배경 단색 #00FF00
- [ ] ✦ 워터마크 없음 — 있으면 `char/remove_gemini_watermark.py`로 제거(보통 우하단 프레임)
- [ ] 좀비처럼 프레임 차이가 크면 워터마크 도너 정합이 나쁠 수 있음 → `--force` 후 눈검증, 정 안 되면 손보정
